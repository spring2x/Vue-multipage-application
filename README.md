# vue-webpack-multipage
webpack多页打包集成vue开发环境

# [项目网址（www.31huiyi.com）][1]

# 项目介绍
*  [31会议官网][2]
*  基于Vue2.0
*  使用iview组件库
*  使用webpack打包
*  前后端分离
# 开始项目
*  npm install
*  npm start 启动服务 localhost:8888/static/apps/home开发调试
*  npm run build 打包发布 localhost:8888/home预览
